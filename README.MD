# TMGDL

Text mode game development library for Borland/TMT Pascal. 
NOTE: This is a work in progress.

## Games included:

- 2048 (updated version from previous repo)
- Game of Life
- Tic-Tac-Toe (Player vs Player or Player vs AI)
- Tetris
